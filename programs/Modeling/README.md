# 3D Modeling Program (Bubble)

**Your professional STL manipulation and cookie cutter generation tool**

## What This Program Does

This is your main 3D modeling application for creating and modifying STL files. Originally built for cookie cutter generation, it's grown into a full STL editing suite.

## Core Features

### Cookie Cutter Generation
- Upload any image (PNG, JPG, etc.)
- Auto-detects outline using AI edge detection
- Generates professional cookie cutters
- Perfect for selling on Etsy!

### STL Editing Tools
- **Widen Hole** - Enlarge cylindrical holes (pain beads, fidget pens) ⭐
- **Thicken** - Make walls thicker for stronger prints ⭐
- **Hollow** - Hollow out models to save filament
- **Repair** - Fix mesh issues automatically
- **Simplify** - Reduce polygon count
- **Mirror** - Mirror across any axis
- **Scale** - Resize models
- **Boolean** - Combine/subtract meshes
- **Split/Cut** - Cut models for large prints
